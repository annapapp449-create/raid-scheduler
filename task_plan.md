# 任务计划 - WoW 时光服开团报名工具

## 项目概述

**项目名称**：时光服开团报名工具
**项目类型**：微信 H5 应用（无需登录注册）
**核心功能**：团长发布开团计划 → 玩家扫码一键报名
**技术栈**：Vite + React 19 + Tailwind CSS v4 + LeanCloud + React Router v7 (Hash)
